An open-source Discord clone. Optimized for security, stability, safety, and customization.

## The Stack

The app is built with:
- Backend
    - Rust
    - Axum
    - Tokio
    - SeaORM
    - Serde
- Frontend:
    - TypeScript
    - Vite
    - SvelteKit
    - SASS
